Παρουσίαση Πτυχιακής

Το project επιτρέπει στον χρήστη να ανεβάζει ευρεσιτεχνίες (.zip ή 7z), σε μια βάση δεδομένων (MySQL) μέσω της αρχικής σελίδας, διαχειριζόμενος φακέλους και προβάλλοντας όσους έχουν ήδη καταχωρηθεί. Στη συνέχεια, μέσω του UI της βάσης, μπορεί να δει αναλυτικά τα δεδομένα, ενώ παρέχεται και στατιστική ανάλυση των καταχωρίσεων. Επιλέξτε την αρχική σελίδα για την εισαγωγή των δεδομένων.

Βασικοί στόχοι πτυχιακής εργασίας
🎯Στόχος του Project
Η εφαρμογή στοχεύει στη διαχείριση, ανάλυση και παρουσίαση δεδομένων με τρόπο που υποστηρίζει την πτυχιακή εργασία και αυτοματοποιεί μεγάλο μέρος της διαδικασίας αξιολόγησης.
📥Τι δεδομένα συλλέγει ο χρήστης
Ο χρήστης ανεβάζει αρχεία δεδομένων xml που περιέχουν παραμέτρους ή αποτελέσματα πειραμάτων, τα οποία αποθηκεύονται στη βάση για περαιτέρω επεξεργασία.
⚙️Τι αναλύσεις εκτελούνται
Το σύστημα πραγματοποιεί καθαρισμό δεδομένων, στατιστική επεξεργασία, ομαδοποίηση τιμών, δημιουργία σεναρίων ανάλυσης και αξιολόγηση της ποιότητας των δεδομένων.
🧠 Τι τεχνολογίες χρησιμοποιούνται
Η πλατφόρμα χρησιμοποιεί Python και Flask για επεξεργασία και διαχείριση δεδομένων, HTML/CSS/JavaScript για το διαδραστικό περιβάλλον, SQL βάση για αποθήκευση, και βιβλιοθήκες όπως Pandas, NumPy και Chart.js για στατιστική ανάλυση και δημιουργία οπτικοποιήσεων.

Σύντομη περιγραφή

Κάθε γράφημα συνοδεύεται από λεκτική ερμηνεία, ώστε η επιτροπή να μπορεί να συνδέσει ποσοτικά και ποιοτικά ευρήματα.

Πώς βοηθά η εργασία;
  • Παρέχει μια ολοκληρωμένη πλατφόρμα για εισαγωγή, επεξεργασία και ανάλυση δεδομένων.
  • Αυτοματοποιεί διαδικασίες που διαφορετικά θα απαιτούσαν χειροκίνητη εργασία.
  • Βελτιώνει την ακρίβεια και την ταχύτητα παραγωγής στατιστικών αποτελεσμάτων.
  • Διευκολύνει τη δημιουργία γραφημάτων και οπτικοποιήσεων για παρουσίαση.
  • Οργανώνει τα δεδομένα με ασφαλή και δομημένο τρόπο σε SQL βάση.
  • Ενισχύει την κατανόηση των αποτελεσμάτων μέσω καθαρής και φιλικής διεπαφής.
